# Kit de Eventos SESC

Repositório modular para gestão de eventos SESC, iniciado com o **Fecomércio 2026**.

## Eventos atuais
- [Fecomércio 2026](./evento-fecomercio-2026/README.md)

## Estrutura
- `evento-fecomercio-2026/`: projeto isolado do evento
- `shared/`: componentes, estilos e scripts reutilizáveis
- `templates/`: base para criação de novos eventos
- `assets-globais/`: mídias compartilhadas

## Como começar
1. Consulte `EVENTOS-ROADMAP.md`
2. Leia `evento-fecomercio-2026/docs/LEIA-ME-PRIMEIRO.md`
3. Use `templates/evento-template/` para novos eventos
